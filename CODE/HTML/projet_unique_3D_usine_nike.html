3D : NIKE'S FACTORY

Date : 12/2024

DESCRIPTION

Student project in 3D modeling and animation: Velocitimac


I chose to model and animate the final stages of sneaker production in a Nike factory. I opted for a sleek style, in line with the modern and minimalist image conveyed by the Nike brand.

The shoe is placed on a conveyor belt where it undergoes various transformations.

I also incorporated movement in the background by modeling the assembly line production process while showcasing the journey of each object before it reaches the scene.


To make the animation more dynamic, I wanted to add transformations to the main object, such as changing the color of the Nike logo on the sneaker and placing it inside its shoebox. To achieve this, I modeled three different robots: one that stamps the Nike logo in blue, one that places the sneaker in the box, and one that closes the box lid. External assets were used for textures (from Poly Haven).

- 2 types of leather textures for the shoe

- 1 concrete texture for the walls

- 1 tile texture for the floor


SOFTWARE

Blender
